
## Install packages

```bash
    npm install
    # or
    yarn add
    # or
    pnpm install
    # or
    bun install
```

---

## Run the deploy server: 

```bash
    **npm**
    npm run build
    npm start

    # or

    **yarn**
    yarn build
    yarn start

    # or

    **pnpm**
    pnpm run build
    pnpm start

    # or

    **bun**
    bun run build
    bun run start

```
